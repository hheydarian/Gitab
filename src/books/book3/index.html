Clean Code

یادگیری اصول نوشتن کد تمیز، ساده و قابل نگهداری

CleanCode# Refactoring# UncleBob# SoftwareDesign# CodeSmells#

کتاب Clean Code نوشتهٔ Robert C.Martin (عمو باب) یکی از مراجع بنیادین در مهندسی نرم‌افزار است. این کتاب مفاهیم عمیق کدنویسی تمیز را به زبانی ساده و عملی آموزش می‌دهد و تمرکز آن روی ایجاد کدی است که خوانا، تست‌پذیر و نگهداشت‌پذیر باشد.

✅ نسخه: 2008

✍️ وضعیت ترجمه: تکمیل شده
